
# BOOSE Web Deployment - MyWebApp

This project is a web-based implementation of the **BOOSE (Basic Object Oriented Software Environment)**. It features a custom interpreter that parses text commands to generate graphical output and manage logical programming structures like loops and variables through an ASP.NET Core web interface.

To get a local copy up and running, follow these steps:

### Prerequisites

* **.NET 8.0 SDK** or later.
* **libgdiplus** (If running on macOS or Linux).

### Installation & Setup

1. **Clone the repository**
```bash
git clone https://github.com/Prabesh666/Ase-Web-Deployment-

```


2. **Navigate to the web application folder**
```bash
cd Ase-Web-Deployment-/MyWebApp

```


3. **Restore NuGet packages**
```bash
dotnet restore

```


4. **Build and Run**
```bash
dotnet run

```


Once the application starts, navigate to the URL provided in the terminal (usually `http://localhost:5000` or `https://localhost:5001`)
